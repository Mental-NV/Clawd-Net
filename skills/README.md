# ClawdNet Skills

These skills are local operating playbooks for high-value migration workflows in this repo.

- `cli-parity-audit`
  - inspect the legacy TypeScript CLI and current .NET CLI, then update parity status accurately
- `migration-doc-sync`
  - keep the stable docs aligned with implementation and migration state
- `dotnet-cli-validation`
  - run the required .NET validation flow sequentially and classify failures clearly
- `interactive-flow-migration`
  - analyze a legacy Ink flow and map it to the .NET TUI or a simpler replacement

Each skill lives in its own folder with a `SKILL.md` file.
